# 💰 2️⃣ Cost constraint (VERY IMPORTANT)

👉 PMD:

* ₹10k – ₹80k range devices

👉 அதனால்:

➡️ design goal:

* cheap components
* efficient hardware

---

# 🔋 3️⃣ Power & Energy (CORE 🔥)

👉 biggest constraint:

➡️ **battery life**

---

## ⚠️ Why power important?

* battery limited
* fan இல்ல ❌
* plastic body → heat problem

👉 அதனால்:

➡️ CPU low power ஆக இருக்கணும்

---

# 💾 4️⃣ Storage choice

👉 PMD use பண்ணுது:

➡️ **Flash memory**

❌ Hard disk இல்ல

👉 காரணம்:

* small size
* low power
* fast enough

---

# 🧠 5️⃣ Embedded vs PMD difference

👉 PMD:

* apps install பண்ணலாம் (Play Store 😄)
* third-party software run ஆகும்

👉 Embedded:

* fixed function மட்டும்

---

## 💥 Important line

👉
**“PMD = flexible platform”**
**“Embedded = fixed device”**

---

# ⏱️ 6️⃣ Real-time performance

* video play 🎥

👉 condition:

➡️ each frame on time process ஆகணும்

## Types:

# 💾 8️⃣ Memory importance

👉 memory:

* power consume பண்ணும்

👉 அதனால்:

➡️ optimize பண்ணணும்

## 💥 Code size importance

👉 reason:

* memory = expensive

**small code = better**

---

# ⚖️ 9️⃣ PMD Design priorities

👉 main goals:

* 🔋 energy efficiency
* 💰 low cost
* ⚡ responsiveness
* 💾 memory optimization

---

# 💥 One-line summary

👉
**“PMD = battery-based, cost-sensitive, user-focused computing system”**

👉 PMD design = hardest balance:

* performance vs battery
* cost vs quality
* speed vs heat
