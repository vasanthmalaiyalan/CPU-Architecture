👉 Simple Tamil:

> “எண்ணிக்கை (numbers) மேல operations செய்யும் circuits”

---

## 💥 Computers என்ன operations செய்கிறது?

இந்த section சொல்லுறது 👇

### முக்கிய operations:

* ➕ Addition (கூட்டல்)
* ➖ Subtraction (கழித்தல்)
* ⚖️ Comparison (பெரியது/சிறியது check)
* 🔁 Shift (bit move)
* ✖️ Multiplication
* ➗ Division

👉 இவை எல்லாம் CPUல daily நடக்குது

---

## ⚙️ இது ஏன் முக்கியம்?

👉 காரணம்:

> 👉 CPU-வின் “brain” = Arithmetic Logic Unit (ALU)

👉 ALU உள்ளே இந்த circuits தான் இருக்கும்

## ➖ 2. Subtraction

👉 Direct subtract பண்ண மாட்டாங்க 😲

👉 Trick use பண்ணுவாங்க:

> Subtraction = Addition + 2’s complement

👉 Tamil:

> “கழித்தல் கூட கூட்டலா தான் செய்யப்படுகிறது”

👉 Example:

* A > B ?

## ✖️ 5. Multiplication

👉 Repeated addition concept

Example:

```
3 × 4 = 3 + 3 + 3 + 3
```

👉 Hardwareல optimized version use பண்ணுவாங்க

👉 Tamil:

> “மீண்டும் மீண்டும் கூட்டுவது”

---

## ➗ 6. Division

👉 Repeated subtraction concept

Example:

```
12 ÷ 3 = subtract 3 repeatedly
```

👉 Hardwareல fast algorithms use பண்ணுவாங்க

👉 Tamil:

> “மீண்டும் மீண்டும் கழிப்பது”
