# 🧠 1.4.4 Bytes, Nibbles, and All That Jazz — Main idea

👉
🔥 **Data எப்படி group பண்ணுறோம் → bit → byte → word**

---

# 🔢 1. Bit → Byte

## 🔹 Bit

👉 smallest unit:

```text id="5t9p7m"
0 or 1
```

---

## 🔹 Byte (VERY IMPORTANT 🔥)

👉

```text id="3y7s1k"
1 byte = 8 bits
```

👉 possibilities:

```text id="b6b6qf"
2⁸ = 256 values
```

---

## 🎯 Example

```text id="j6hj5q"
01000001 → 'A'
```

👉 ASCII character 🔥

---

# 🧩 2. Nibble

👉

```text id="w2jxxt"
1 nibble = 4 bits
```

👉 possibilities:

```text id="f6i7cw"
2⁴ = 16
```

---

## 🎯 Connection

👉

🔥 **1 hex digit = 1 nibble**

👉

```text id="l9n8kw"
FF₁₆ = 11111111₂ (1 byte)
```

---

# 🧠 Summary

| Unit   | Size   |
| ------ | ------ |
| Bit    | 1      |
| Nibble | 4 bits |
| Byte   | 8 bits |

---

# 🧠 3. Word (CPU level 🔥)

👉

🔥 **CPU handle பண்ணுற data size = word**

---

## 🔹 Types

| CPU           | Word size  |
| ------------- | ---------- |
| Old           | 32-bit     |
| Modern        | 64-bit 😎  |
| Small devices | 8 / 16-bit |

---

👉 example:

👉 64-bit CPU:

```text id="0zj7fs"
1 word = 64 bits
```

---

# 🧠 4. LSB vs MSB (VERY IMPORTANT 🔥)

---

## 🔹 LSB (Least Significant Bit)

👉 rightmost bit

👉 smallest value (1’s place)

---

## 🔹 MSB (Most Significant Bit)

👉 leftmost bit

👉 biggest value

---

## 🎯 Example

```text id="2t71ts"
101100
```

| Bit       | Role |
| --------- | ---- |
| Rightmost | LSB  |
| Leftmost  | MSB  |

---

# 🧠 Bytes also same

👉 4-byte number:

👉 first byte = LSB
👉 last byte = MSB

---

# ⚡ Why important?

👉

✔️ memory layout
✔️ endianness
✔️ exploit development 🔥

---

# 🧠 5. Powers of Two shortcut

👉

🔥 **2¹⁰ ≈ 1000**

---

## 🔹 Units

| Name | Value               |
| ---- | ------------------- |
| KB   | 2¹⁰ ≈ 1,000         |
| MB   | 2²⁰ ≈ 1,000,000     |
| GB   | 2³⁰ ≈ 1,000,000,000 |

---

## 🎯 Exact values

```text id="u7nlkk"
1 KB = 1024 bytes  
1 MB = 1024 KB  
1 GB = 1024 MB
```

---

# 🧮 Example 1.6

👉 find:

```text id="1c3r6z"
2²⁴
```

---

## 🔹 Solve

```text id="m1jvsl"
2²⁴ = 2²⁰ × 2⁴  
≈ 1 million × 16  
≈ 16 million ✅
```

---

## 🔍 Exact:

```text id="v9zgjm"
16,777,216
```

---

## 🔹 Memory

---

## 🔹 Network speed


## 🎯 Example


# ⚡ Important difference

```text id="h3fdre"

```

👉 8 Mbps ≠ 8 MBps 😅
