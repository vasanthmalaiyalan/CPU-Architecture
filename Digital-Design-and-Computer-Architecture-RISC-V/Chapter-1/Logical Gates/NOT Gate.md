# 🎯 Definition

👉

✔️ 1 input → A
✔️ 1 output → Y

👉

🔥 **Output = input opposite**

---

# 🧩 Truth Table

| A | Y |
| - | - |
| 0 | 1 |
| 1 | 0 |

---

# 🧠 Boolean Equation

```text id="d5j8k7"
Y = NOT A
```

👉 book notation:

```text id="v4k3lw"
Y = A̅
```

---

# 🔤 Different notations (important 🔥)

👉 same meaning:

```text id="r9gk2p"
Y = A̅  
Y = A'  
Y = ¬A  
Y = !A  
Y = ~A
```

👉

🔥 எல்லாம் “NOT A” தான்

---

# 🧠 Example

👉

```text id="3bn4gx"
A = 0 → Y = 1  
A = 1 → Y = 0
```

---

# ⚡ Why important?

👉
