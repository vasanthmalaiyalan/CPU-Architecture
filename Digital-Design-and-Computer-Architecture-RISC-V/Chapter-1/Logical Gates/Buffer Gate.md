# 🧠 What buffer actually does?

---

## 🔹 1. Current increase

👉

✔️ small signal → big signal

👉

🔥 motor / LED drive பண்ண முடியும்

---

## 🔹 2. Fan-out (multiple outputs)

👉

✔️ ஒரு signal → பல gatesக்கு send

👉

🔥 signal weak ஆகாமல்

---

## 🔹 3. Speed improvement

👉

✔️ fast signal transmission

---

# 🧠 Example

👉

```text
CPU signal → directly 100 gatesக்கு போனால் → weak 😵
```

👉

🔥 buffer use பண்ணினா:

```text
CPU → Buffer → 100 gates (strong signal) ✅
```
