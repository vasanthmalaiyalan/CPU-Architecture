# 🧠 1.4 NUMBER SYSTEMS — Main idea

👉
🔥 **Computer → decimal இல்ல → binary (0/1) use பண்ணும்**

---

# 🎯 Why different number systems?

👉 நாம் use பண்ணுவது:

✔️ Decimal (base 10)

👉 computer use பண்ணுவது:

✔️ Binary (base 2)

👉 engineers use பண்ணுவது:

✔️ Hexadecimal (base 16)

---

# 🧩 1. Decimal (Base 10)

👉 digits:

```text id="5efy3x"
0 – 9
```

👉 example:

```text id="llr2y7"
345 = 3×10² + 4×10¹ + 5×10⁰
```

---

# 🧩 2. Binary (Base 2) 🔥

👉 digits:

```text id="r3lh4f"
0, 1 மட்டும்
```

---

## 🔹 Example

```text id="2h4eqd"
1011₂
```

👉 convert பண்ணலாம்:

```text id="0c6c4g"
= 1×2³ + 0×2² + 1×2¹ + 1×2⁰  
= 8 + 0 + 2 + 1  
= 11 (decimal)
```

---

# 🧠 Important

👉

🔥 **CPU எல்லா calculationவும் binaryல தான்**

---

# 🧩 3. Hexadecimal (Base 16) 🔥

👉 digits:

```text id="zbx3c8"
0–9 + A–F
```

| Hex | Value |
| --- | ----- |
| A   | 10    |
| B   | 11    |
| C   | 12    |
| D   | 13    |
| E   | 14    |
| F   | 15    |

---

## 🔹 Example

```text id="bo1x7a"
2F₁₆
```

👉 convert:

```text id="yzr9ow"
= 2×16¹ + 15×16⁰  
= 32 + 15  
= 47 (decimal)
```

---

# 🔥 Why Hex important?

👉 binary:

```text id="2w4ynf"
11110000 😵
```

👉 hex:

```text id="3wiy6a"
F0 😎
```

👉

✔️ short
✔️ readable

---

# 🧠 Binary ↔ Hex relation

👉

🔥 **1 hex digit = 4 bits**

---

## 🔹 Example

```text id="24g23a"
1111 → F  
1010 → A
```

---
