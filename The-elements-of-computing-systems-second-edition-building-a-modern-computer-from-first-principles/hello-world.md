# 💻 Hello World – Real Meaning (Simple Tamil)

---

## 🧾 1. Program என்ன?

நீ பார்க்குற code 👇

```
class Main {
   function void main() {
      do Output.printString("Hello World");
      return;
   }
}
```

👉 இது என்ன செய்கிறது?

👉
**"Hello World" print பண்ணி exit ஆகும்**

---

👉 இது:
👉
**High-Level Language (Jack)**

# 🤯 2. Reality Check (மிக முக்கியம்)

👉 இந்த program:

👉
**Computerக்கு direct புரியாது ❌**

👉 Computer புரியும் language:

**Machine Code (0 & 1) மட்டும்**

# 🔄 3. What actually happens? (Step-by-step)

---

## 📝 Step 1: Text File

👉 இந்த program:

👉
**just characters தான்**

```
H e l l o W o r l d
```

👉 Computerக்கு இது meaning இல்ல ❌

---

## 🧠 Step 2: Parsing

👉 Compiler:

* code read பண்ணும்
* meaning புரியும்

👉 example:

* printString → output செய்யணும்

---

👉 simple:
👉
**"Code-ஐ புரிஞ்சிக்கிறது"**

---

## 🔄 Step 3: Compilation

👉 High-level code → machine code

👉 இது தான்:
👉
**Compilation**

---

👉 output:

```
1010101010...
```

👉 CPUக்கு புரியும் format ✅

# ⚙️ 4. Machine Code கூட abstraction தான்

👉
**binary rules set (agreement)**

👉 அதை run பண்ண:

👉 hardware வேண்டும்

👉
**0 / 1 physical form**
